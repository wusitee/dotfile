#+title: Computer Setup Guide

* Overview
This document contains the complete setup guide for a personal Arch Linux system running on a Huawei MateBook X Pro 2021. It covers hardware-specific configurations, system setup, desktop environment configuration, and application installation.

The system uses Hyprland as the primary window manager with a suite of Wayland-native tools for daily computing.

* Hardware-Specific Configuration

** Battery Management (MateBook)
TLP is the power management tool of choice:
#+begin_src shell
systemctl enable tlp --now
#+end_src

In addition to TLP, we also use matebook-applet and huawei-wmi for battery threshold control:
#+begin_src shell
yay -S matebook-applet huawei-wmi
#+end_src

Additionally, use the script in dotfiles-huawei-wmi to restore battery threshold after suspension.

** Sound (MateBook)
Install the firmware:
#+begin_src shell
sudo pacman -S sof-firmware
#+end_src

*** Troubleshooting
**** No sound after suspend (MateBook specific)
Follow [[https://github.com/codepayne/linux-sound-huawei/issues/15][this GitHub issue]] to fix sound issues on Huawei MateBook devices.

* Base System

** Boot
intel-ucode for Intel processors:
#+begin_src shell
sudo pacman -S intel-ucode
#+end_src

** Firewall
Using nftables and ufw:
#+begin_src shell
sudo pacman -S iptables-nft ufw
#+end_src

Default configuration for [[https://wiki.archlinux.org/title/Uncomplicated_Firewall][ufw]]:
#+begin_src shell
sudo ufw default deny
sudo ufw allow from 192.168.0.0/24
sudo ufw allow Deluge
sudo ufw limit ssh
sudo ufw enable
#+end_src

* Desktop Environment

** Hyprland (Primary)
[[https://wiki.archlinux.org/title/Hyprland][Hyprland]] is a dynamic tiling Wayland compositor.

Using xdg desktop portal:
#+begin_src shell
sudo pacman -S xdg-desktop-portal-hyprland xdg-desktop-portal-gtk
#+end_src

*** Status Bar (waybar)
Using waybar for status bar:
#+begin_src shell
sudo pacman -S waybar
#+end_src

*** Launcher (rofi)
Using rofi as application launcher:
#+begin_src shell
sudo pacman -S rofi-wayland
#+end_src

And in hyprland.conf:
#+begin_src conf
$menu = rofi -show combi
#+end_src

*** Clipboard (copyq)
Using copyq as clipboard manager:
#+begin_src shell
sudo pacman -S copyq
#+end_src

*** Screenshots (hyprshot)
Use hyprshot for screenshots:
#+begin_src shell
yay -S hyprshot
#+end_src

*** Notifications (swaync)
Notification center:
#+begin_src shell
sudo pacman -S swaync
#+end_src

*** Idle/Lock (hypridle/hyprlock)
Screen lock and idle management:
#+begin_src shell
sudo pacman -S hyprlock hypridle
#+end_src

*** Wallpaper (awww)
Using awww (successor to swww) for animated wallpapers:
#+begin_src shell
yay -S awww
#+end_src

Add to hyprland.conf autostart:
#+begin_src conf
exec-once = awww-daemon
#+end_src

*** Theming (GTK, Qt)
GTK theme is managed via config files in the dotfiles repository:
- =gtk-3.0/settings.ini= — Uses =Adwaita-dark= (GTK3)
- =gtk-4.0/settings.ini= — Uses =Adwaita= with =prefer-dark-theme= (GTK4)

Copy these to =~/.config/= to apply:
#+begin_src shell
cp gtk-3.0/settings.ini ~/.config/gtk-3.0/settings.ini
cp gtk-4.0/settings.ini ~/.config/gtk-4.0/settings.ini
#+end_src

Qt theming is set via =qt6ct=:
#+begin_src shell
sudo pacman -S qt6ct
#+end_src

*** Default Applications
Use xdg-mime to change the default program that opens a specific type. For example to set default PDF reader to okular:
#+begin_src shell
xdg-mime default okularApplication_pdf.desktop application/pdf
xdg-mime query default application/pdf  # should be "okularApplication_pdf.desktop"
#+end_src

**** Hide the horizontal tabs (Sideberry CSS)
:PROPERTIES:
:ID:       95658cd2-b982-4efb-a472-bea9a70870ea
:END:
Do [[https://github.com/FirefoxCSS-Store/FirefoxCSS-Store.github.io/blob/main/README.md#generic-installation][the following]], and in userChrome.css:
#+begin_src css
/* hide the tabs on the top of the window */
:root{
    --sidebar-hover-width: 22px;
    --sidebar-visible-width: 140px;
}
#TabsToolBar {
    visibility: collapse;
}
#TabsToolbar, #sidebar-header{
    display: none !important;
}

#sidebar-box {
    position: relative !important;
    overflow: hidden;
    max-width: var(--sidebar-hover-width) !important;
}

#sidebar-box:hover {
    transition: all 200ms !important;
    max-width: var(--sidebar-visible-width) !important;
}
#+end_src

**** Chinese Language Pack
#+begin_src shell
sudo pacman -S firefox-i18n-zh-cn
#+end_src

**** Troubleshooting: When Firefox is laggy
You can choose to refresh Firefox and redo the above configuration.

**** Troubleshooting: Firefox extension popups are oversized on Wayland
On Firefox 150 with Wayland/HiDPI, extension popups such as uBlock Origin can
render extremely large and stop registering clicks correctly. The workaround
from this [[https://bbs.archlinux.org/viewtopic.php?id=313261][Arch Linux forum thread]] is:

- Open =about:config=.
- Set =widget.wayland.fractional-scale.enabled= to =false=.

** CLI Applications

*** Shell (Fish)
Using fish as the interactive shell:
#+begin_src shell
sudo pacman -S fish
#+end_src

*** Autojump (zoxide + fzf)
Using zoxide for directory jumping:
#+begin_src shell
sudo pacman -S zoxide fzf
zoxide init fish
#+end_src

*** File Manager (yazi)
Terminal file manager with drag-and-drop and clipboard support:
#+begin_src shell
sudo pacman -S yazi parallel ripdrag xclip
#+end_src

Plugins (managed via =ya pkg=):
#+begin_src shell
ya pkg add boydaihungst/restore
ya pkg add simla33/ucp
#+end_src

Key features:
- Press =C-n= in Yazi to drag and drop selected files
- Press =C-y= in Yazi to copy selected files to system clipboard (Wayland + XWayland via =ucp.yazi=)
- Files open via =xdg-open= using GNU Parallel for batch opening
- Fish integration: =y= command changes cwd on exit

* Dotfiles Management
This repository contains configuration files for the entire system. Here's how to apply changes:

** Applying Configuration Changes
| Component      | Reload Command                              |
|----------------+---------------------------------------------|
| Hyprland       | =hyprctl reload= (or auto-reloads on save)  |
| Doom Emacs     | =doom sync= (after init.el/packages.el changes) |
| Fish           | =source ~/.config/fish/config.fish=         |
| Waybar         | =killall waybar && waybar=                  |

** Repository Structure
- =hypr/= - Hyprland window manager configuration
- =doom/= - Doom Emacs configuration (edit =config.org=, not =config.el=)
- =fish/= - Fish shell configuration
- =ghostty/= - Ghostty terminal configuration
- =waybar/= - Status bar configuration
- =rofi/= - Application launcher configuration
- =yazi/= - Terminal file manager configuration
- =gtk-3.0/= - GTK3 theme configuration
- =gtk-4.0/= - GTK4 theme configuration
